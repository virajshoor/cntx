Built for repeated work

Agentic coding features with lower install friction.

Sandboxed apply mode

The model emits complete fenced blocks with path=. Cntx previews the changes, writes them through a project-root sandbox, and prints a file checklist.

Interactive markdown chat

Assistant output, help, inline code, and lists render cleanly in the terminal while a dot animation shows live work.

Bring your own keys

Keys stay in a local runtime store, outside source and outside the published crate.

Model routing

Optimized prompt size routes work to small, balanced, or stronger model families.

MCP-ready

Built-in Context7 doc search and Headroom token saving are configured as on-demand MCP servers.

Counsel mode

A bounded evaluator pass helps choose the right worker model without multiplying context blindly.

Quick start

Install, add a key, and run the first prompt.

Cntx keeps setup plain: install the binary, store a provider key, create an endpoint, then start chatting or apply file edits.
